# P0 Brainstorm Report: ClawAppStore

> Generated: 2026-03-15
> Idea: "AI专属的AppStore，提供各种供AI Agent调用的通用软件包，按次收费or订阅"

---

## 1. 产品概念

### 一句话定义

**AI Agent 专属的工具市场——开发者发布工具包，AI Agent 自主发现、调用、付费。**

### 详细描述

一个面向 AI Agent 的工具/API 市场平台。开发者（供给侧）将软件包/API/MCP Server 发布到平台，AI Agent 及其开发者（需求侧）通过统一协议发现并调用这些工具。平台提供发现、认证、计量、计费一站式服务，按次收费或订阅制。

### 核心问题

AI Agent 需要调用外部工具完成任务（搜索、代码执行、数据处理、通信等），但当前工具生态碎片化严重：
- 发现困难：工具分散在 GitHub、npm、各种目录站
- 接入成本高：每个工具有不同的认证、格式、调用方式
- 无统一计费：Agent 调用 10 个工具需要 10 个账号和付费方式
- 质量不透明：没有标准化的可靠性、性能、安全评估

### 目标用户

1. **AI Agent 开发者**（需求侧）：使用 LangChain/CrewAI/AutoGen 等框架构建 Agent 的开发者
2. **工具发布者**（供给侧）：拥有 API/服务/数据的开发者，希望将其 Agent 化并变现
3. **企业用户**：运行多 Agent 系统的公司，需要统一的工具管理和计费

### 市场趋势

### 时机分析

**窗口期 12-18 个月**。AI Agent 生态正处于从碎片化到标准化的转折点，MCP 正在统一协议层，但市场/商业层还没有明确的赢家。大厂（OpenAI/Anthropic/Google）18 个月内大概率会推出自己的工具商店。

## 4. 技术可行性

### 推荐技术栈

| 层 | 选型 | 理由 |
|----|------|------|
| Frontend | Next.js 15 | ClawLabz 统一栈，SSR/SSG 灵活 |
| Backend API | Next.js API Routes + Edge Functions | Vercel Serverless，低延迟 |
| Database | Supabase PostgreSQL | 已有基础设施 |
| Auth | Supabase Auth + API Key | 开发者 API Key 认证 |
| Proxy/Gateway | Vercel Edge / Cloudflare Workers | 工具调用代理层，全球边缘 |
| 计量 | 自建（PostgreSQL + Redis） | 按次计量，实时统计 |
| 支付 | Stripe (法币) + CLAW (链上) | 双轨支付 |
| 协议 | MCP 原生 + OpenAI Function Calling 适配 | 覆盖主流 Agent 框架 |

### 技术风险

| 风险 | 级别 | 应对 |
|------|------|------|
| 工具调用代理的延迟 | 中 | Edge Function 就近路由，<50ms 代理开销 |
| 计量准确性 | 中 | 异步计量 + 对账，最终一致 |
| 工具安全性（恶意工具） | 高 | 沙箱执行 + 审核机制 + 信任分级 |
| 跨协议适配（MCP/OpenAI/自定义） | 中 | 统一中间协议层 + 适配器模式 |

### 复杂度评估

**中等偏高** — 核心难点在代理层（低延迟、高可靠）和计量计费系统。前端和市场展示部分较标准。

## 6. 商业闭环

### 收入模型

**双轨收入：平台抽成 + 增值服务**

| 收入来源 | 模式 | 预估 |
|---------|------|------|
| 交易抽成 | 工具调用收入的 20% | 核心收入 |
| 免费额度广告 | 工具发布者可购买推荐位 | 补充收入 |
| 企业版 | 私有化部署、SLA 保障、合规审计 | 高客单价 |
| CLAW 支付优惠 | 用 CLAW token 支付享折扣，驱动 token 流通 | 生态价值 |

### 定价策略

```
Free Tier:     每个工具 100 次/月免费调用
Pay-per-use:   超出后按次计费（每次 $0.001 - $0.10，由工具定价）
Volume Pack:   预购调用包，享 20-40% 折扣
Subscription:  无限调用特定工具，$X/月
```

### 单位经济

| 指标 | 数值 |
|------|------|
| 平台代理成本/次 | $0.00005 - 0.0003 |
| 平均工具定价/次 | $0.001 - 0.01 |
| 平台收入/次 (20% 抽成) | $0.0002 - 0.002 |
| 毛利率 | 50-85% |
| 目标用户 ARPU (Year 1) | $15-40/月 |
| Free → Paid 转化率 (行业基准) | 2-5% |

### 成长路径

1. **冷启动** — 预装 50-100 个高质量工具（搜索/代码执行/数据处理/通信），覆盖 Agent 最常见需求
2. **开发者激励** — 前 6 个月发布者享 90/10 分成（平台只抽 10%）
3. **SDK 降低门槛** — 提供工具将任意 REST API 一键转为可发布工具包
4. **框架集成** — 直接集成到 LangChain/CrewAI/AutoGen，工具发现内嵌到开发流程
5. **企业拓展** — 统一计费、合规审计、SLA 保障

## 8. 结论：PIVOT

### 推荐方向调整

原始 idea "AI 专属 AppStore" 太通用，直接对标 Composio ($29M) 和未来的 OpenAI/Anthropic 官方商店，胜算不高。

**PIVOT 为：ClawToolkit — OpenClaw 生态的 AI Agent 工具协议与市场**

核心调整：

| 原始方向 | 调整后方向 |
|---------|-----------|
| 通用 AI AppStore | **OpenClaw 生态工具市场** — 差异化定位 |
| 纯中心化平台 | **中心化平台 (MVP)** — Supabase 数据库，后期可选链上存证 |
| 纯 CLAW 支付 | **法币为主 + CLAW 为辅** — Stripe 零摩擦入口，CLAW 折扣激励 |
| 和所有人竞争 | **MCP-native + 多协议适配** — 站在 MCP 标准之上而非之外 |
| 独立新产品 | **OpenClaw 六大产品联动** — Arena/Market/Genesis 的 Agent 天然成为需求方 |

### 调整理由

1. **生态联动优势**：ClawArena 的 NPC Agent、ClawMarket 的任务 Agent、ClawGenesis 的文明 Agent 都需要调用工具，天然的需求侧冷启动
2. **MCP 兼容而非对抗**：不替代 MCP 生态，在其之上加商业层和支付层
3. **避免正面竞争**：不和 Composio ($29M) 或未来 OpenAI 商店正面竞争通用市场，做 OpenClaw 生态的垂直工具市场

### 关键架构决策：不上链 + 法币为主

#### 决策一：MVP 不上链

- 所有成功工具市场 (npm, RapidAPI, Composio) 都是中心化数据库
- 上链注册增加复杂度/Gas 费/延迟，目标用户 (AI 开发者) 不关心去中心化
- PostgreSQL + 审计日志在实际效果上等价于链上不可篡改
- **后期可选**：工具注册哈希锚定到 ClawNetwork 做存证 (v2+)

#### 决策二：三层支付架构

```
用户支付层:    Stripe 法币 (主) + CLAW/USDC (辅, 享 9 折)
              ↓
内部结算层:    CP 积分 (统一计量，隔离汇率波动)
              ↓
分成结算层:    法币 (Stripe Connect) 或 CLAW (链上转账)，发布者自选
```

**CLAW 的角色 — 不是支付门槛，而是激励机制：**
- CLAW 支付享折扣 (驱动 token 需求)
- 质押 CLAW 获更高分成比例
- CLAW 持有者参与工具审核/治理投票

**依据**：纯 Token 支付是已被验证的失败路径 (OpenBazaar 融 $9.25M 后关停)。成功模式是法币为主+crypto为辅 (Ankr, Stripe USDC)。

### MVP 范围建议

1. **工具注册 & 发现**（Web UI + API）
2. **MCP 协议适配器**（一键将 MCP Server 发布到市场）
3. **统一调用代理**（一个 API Key 调用所有工具）
4. **CP 积分计量计费**（复用已有积分系统，Stripe 法币入口）
5. **开发者 Dashboard**（调用统计、收入查看）

### 命名建议

- **ClawToolkit** — 简洁，"工具包"含义直接
- 或直接叫 **ClawStore** — 更符合 AppStore 的原始概念
- 路由: `toolkit.clawlabz.xyz` 或 `store.clawlabz.xyz`
